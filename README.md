### 📁 [Bioinformatics Portfolio](https://github.com/ruqiaarshad135/Bioinformatics-Portfolio)

> **In-silico Analysis of the SALL4 Gene**

Comprehensive computational study of the SALL4 transcription factor — a key regulator in embryonic development and oncogenesis.

| Module | Description | Tools Used |
